Para compilar basta executar:
	g++ -o avl avl.cpp
	g++ -o red_black red_black.cpp

E para executar apenas:
	./avl.exe
	./red_black.exe

*Obs: Para o programa funcionar corretamente, é necessario criar duas pastas, uma chamada "Construcao" e outra "Consulta", 
no mesmo diretorio dos executaveis, e dentro dessas pastas colocar todos os arquivos de construção e consulta respectivamente.
